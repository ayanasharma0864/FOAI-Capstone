# AI Personal Finance Tracker & Advisor

An AI-powered expense tracker designed for students to track, categorize, and analyze spending with smart insights.

## 🚀 New: Google Integration
This project is now fully integrated with Google Forms and Google Sheets:
1. **Google Form**: Log your expenses via a simple, mobile-friendly Google Form.
2. **Google Sheet**: All your data is stored securely in your private Google Sheet.
3. **Real-time Dashboard**: The dashboard automatically fetches data from your sheet and provides AI-powered insights.

## 🛠️ How it Works
1. **Entry**: When you open the dashboard, you'll be prompted to log an expense via the Google Form.
2. **Sync**: Click **Continue to Dashboard** after submission to see your updated stats.
3. **AI Insights**: The system automatically categorizes your spending and provides personalized savings tips.
4. **Local Edits**: Need to correct an amount? Use the ✏️ icon in the "All Expenses" table to make local adjustments.

## 📂 Project Structure
- `index.html`: Main dashboard layout with embedded Google Form.
- `app.js`: Logic for fetching data from Google Sheets CSV and managing UI states.
- `style.css`: Premium dark-themed styling with glassmorphism effects.
